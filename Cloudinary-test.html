How to Embed Your Cloudinary Video

1Upload to Cloudinary

Upload your video to Cloudinary through the dashboard or API. Note your video's public ID.

2Include Required Libraries

Add these lines to your HTML head section to load the Cloudinary Video Player assets.

3Create Video Element

Add a video element with an ID that will be used to initialize the player.

4Initialize the Player

Use JavaScript to initialize the player with your cloud name and video public ID.

Implementation Code

// Include these in your <head> section <link href="https://unpkg.com/cloudinary-video-player@1.9.4/dist/cld-video-player.min.css" rel="stylesheet"> <script src="https://unpkg.com/cloudinary-video-player@1.9.4/dist/cld-video-player.min.js"></script> // HTML video element <video id="player" controls muted class="cld-video-player" width="600" ></video> // JavaScript initialization <script> const player = cloudinary.videoPlayer('player', { cloudName: 'duuhkyscq', publicId: 'istockphoto-1586471923-640_adpp_is_hkkqmn', profile: 'cld-default' }); </script>

Note: Make sure to include the Cloudinary Video Player library before initializing the player. The player will automatically handle responsive sizing and provide a consistent experience across devices.
