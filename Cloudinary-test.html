How to Embed Your Cloudinary Video

1Upload to Cloudinary

Upload your video to Cloudinary through the dashboard or API. Note your video's public ID.

2Use the iframe Embed Code

Copy the iframe embed code provided by Cloudinary or use the code shown below.

3Paste into Your HTML

Insert the iframe code into your HTML file where you want the video to appear.

4Customize as Needed

Adjust the width, height, or add additional parameters to the URL as needed.

Implementation Code

<iframe src="https://player.cloudinary.com/embed/?cloud_name=duuhkyscq&public_id=istockphoto-1586471923-640_adpp_is_hkkqmn&profile=cld-default" width="640" height="360" style="height: auto; width: 100%; aspect-ratio: 640 / 360;" allow="autoplay; fullscreen; encrypted-media; picture-in-picture" allowfullscreen frameborder="0" ></iframe>

Note: The iframe embed method is the recommended way to embed Cloudinary videos as it handles all the complexity of video playback and ensures compatibility across different browsers and devices.

Troubleshooting Tips

If your video isn't playing, try these solutions:

  • Check that your cloud_name and public_id are correct
  • Ensure the video has finished processing on Cloudinary
  • Try using a different browser
  • Check if any browser extensions are blocking the video
  • Verify that your internet connection is working properly

Additional Features

The Cloudinary Video Player supports many advanced features that you can add as parameters to the URL:

  • Autoplay: &autoplay=true
  • Muted: &muted=true
  • Loop: &loop=true
  • Custom controls: &controls=false
  • Video transformations: &transformation=...
